Standard Chaos Risk
Each player is given 20hp (Health points). Every three turns a question will be asked to each player when it is their turn. Upon answering successfully, player will be granted a powerup of their choice. If answering incorrectly, player will be given a random disadvantage. Each player is given 40hp. Every turn a question will be asked to each player. Upon answering successfully, player will be granted a random power up. If answered incorrectly, player will be given a random disadvantage. Each player is given 20hp. Every 3 turns each player can choose from 3 powerups, but will also be given a random disadvantage.
